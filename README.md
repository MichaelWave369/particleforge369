# ParticleForge369

ParticleForge369 is a public, MIT-licensed particle simulation and receipt lab for learning, modeling, and visual exploration.

Core rule: Simulation creates possibility. Real data creates evidence. Repeated validated evidence creates discovery.

This project is independent and not affiliated with CERN, ATLAS, CMS, or any particle physics institution.

## v0.1 focus

- WebGL particle field
- phi / Fibonacci / 369 visual lattice
- Higgs-like educational receipt playground
- toy histogram and signal-bump sketch
- claim status ledger

## Local development

npm install
npm run dev

## Build

npm run build
npm run preview
