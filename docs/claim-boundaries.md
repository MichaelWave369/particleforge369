# Claim Boundaries

ParticleForge369 is an educational and experimental simulation lab.

## Core rule

Simulation creates possibility. Real data creates evidence. Repeated validated evidence creates discovery.

## Allowed claims

- Educational reproduction.
- Toy model only.
- Observed in simulation.
- Interesting residual that needs review.
- Public-data comparison when a dataset and method are cited.

## Forbidden claims

- Replacing real accelerators or detectors.
- Announcing new physics from simulation alone.
- Borrowing authority from CERN, ATLAS, CMS, or any institution.
- Treating visual beauty as evidence.

## Ledger rule

Every run should state what was simulated, what was compared, what data was used, and what the boundary is.
